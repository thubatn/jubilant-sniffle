Salesforce Administrator & Consultant

CRM Customisation & Sales Process Optimisation

Client: ForeverGreens — Retail Company

After initial setup, the sales team identified gaps: missing fields, no win/loss tracking, and a cluttered pipeline. I delivered 9 targeted customisations — from formula-driven commission automation to redesigned page layouts and a streamlined sales process.

Custom Account Fields

Created Active Customer Plan (Checkbox) and Customer Expiration Date (Date) fields for service agreement tracking.

Prospect Rating + Field Security

Renamed Rating to Prospect Rating; restricted editing to Sales Users and System Admins via field-level security.

Win/Loss Reason Picklists

Added custom picklist fields to Opportunity so management can analyse trends and identify patterns in closed deals.

Backup Agent Lookup

Added Backup Agent lookup on Case object to ensure case coverage when the primary agent is unavailable.

Commission Formula Field

Built a formula that auto-calculates 10% commission when Opportunity = Closed Won. Zero manual input required.

High-Value Opportunity View

Custom list view with Kanban visualisation enabling sales reps to focus on their most valuable active deals.

Page Layout Redesign

Added Customer Information section, moved Opportunities related list to top of Account page for faster sales access.

Record Types & Sales Path

Implemented per-account-type record types with separate page layouts; configured Sales Path with coaching text on Qualification.

Sales Process Optimisation

Removed redundant stages (Value Proposition, Perception Analysis) to streamline the pipeline and reduce friction.

03
Project
Salesforce Administrator

Data Management & UX Optimisation

Client: ForeverGreens — Retail Company

ForeverGreens wanted structured customer feedback capture, cleaner record layouts, and efficient bulk data management. I implemented dynamic forms, mobile-optimised layouts, a custom feedback object, and executed large-scale data import, update, and cleanup operations.

Customer Feedback Object

Created custom object with Satisfaction Score (Picklist), Customer Comments (Long Text), and Account Lookup for CRM-linked feedback tracking.

Navigation Optimisation

Built a custom navigation tab for the Customer Feedback object so reps can log and report satisfaction without leaving Salesforce.

Dynamic Forms

Customer Plan Expiration Date only appears when the Active checkbox is ticked — reducing layout clutter for inactive accounts.

Compact Layout (Mobile)

Configured Account compact layout: Name, Number, Site, Status, Revenue, Address, Type, Website — optimised for field reps on mobile.

Lead Data Import

Used Data Import Wizard to upload website leads from CSV, mapped fields, and validated results for immediate sales team use.

Bulk Update via SOQL

Exported accounts via SOQL query, updated Account Site field in dataset, re-imported via Dataloader.io — eliminating record-by-record editing.

Mass Record Deletion

Used Mass Delete to remove incorrectly imported leads (Virtucon), filtered by date and company. Verified via Recycle Bin.

Mass Ownership Transfer

Bulk-transferred accounts, contacts, and open opportunities from Phil Smith → Matt Wilson (CA region) during team restructuring.
